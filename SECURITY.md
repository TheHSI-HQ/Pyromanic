# Security Policy

## Supported Versions

| Version | Supported               |
| ------- | ----------------------- |
| > 1.1.0 | :white_check_mark:      |
| 1.1.0   | :ballot_box_with_check: |
| < 1.1.0 | :x:                     |

- :white_check_mark:: Full Support
- :ballot_box_with_check:: Partial Support
- :x:: Not Supported
